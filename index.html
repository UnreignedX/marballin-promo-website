About

Marballin is a Roguelike game that features marbles and math.

Inspired by Balatro, this game takes on a different twist, where you have to collide marbles and use basic algebra to your advantage to advance to the next stage.

The game features a short assesment on basic algebra, to which you can use to your own advantage. This game also features a leaderboard to compare your previous scores, an in-game shop to buy and purchase upgrades for your cards, and a lot more.

The game is currently in early access, and can undergo a rework or a redesign whenever possible!
