Marballin

Marballin is a Roguelike game that features marbles, and math.

Minimum Specifications

  • Operating System: Windows 10 (64-bit) / macOS 10.15 Catalina / Ubuntu 20.04 LTS
  • Processor: Intel Core i5-7400 or AMD Ryzen 3 1200
  • Memory (RAM): 8 GB
  • Graphics Card: NVIDIA GeForce GTX 960 / AMD Radeon R9 380 (2GB VRAM)
  • DirectX Version: DirectX 11
  • Storage: 200MB available space

Recommended Specifications

  • Operating System: Windows 10 (64-bit) / macOS 11 Big Sur / Ubuntu 22.04 LTS
  • Processor: Intel Core i7-9700K or AMD Ryzen 5 3600
  • Memory (RAM): 8 GB
  • Graphics Card: AMD Radeon RX 5500 XT
  • DirectX Version: DirectX 12
  • Storage: 200MB available space

About

Marballin is a Roguelike game that features marbles, and math.

It is a game where you can play as a marble and fight against other marbles. You can also collect items and power-ups to help you in your journey.
